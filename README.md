# 🛒 Walmart Sales Data Analysis

## 📌 Project Overview

This project analyzes Walmart's weekly sales data using **Python**, **Pandas**, and **NumPy**. The goal is to explore sales performance, identify trends, compare store performance, and understand how external factors such as holidays, fuel prices, temperature, CPI, and unemployment relate to weekly sales.

---

## 📂 Dataset

The dataset contains the following columns:

* Store
* Date
* Weekly_Sales
* Holiday_Flag
* Temperature
* Fuel_Price
* CPI
* Unemployment

## 📊 Project Workflow

### 1. Data Loading

* Imported the Walmart sales dataset.
* Explored the structure of the data.

### 2. Data Cleaning

* Checked for missing values.
* Checked for duplicate records.
* Converted the Date column into datetime format.
* Verified data types.

### 3. Exploratory Data Analysis (EDA)

The following analyses were performed:

* Total Weekly Sales
* Average Weekly Sales
* Monthly Sales Analysis
* Store-wise Sales Analysis
* Top Performing Store
* Holiday vs Non-Holiday Sales
* Correlation Analysis

### 4. Data Visualization

Visualizations include:

* Monthly Sales Trend
* Store-wise Sales Comparison
* Holiday vs Non-Holiday Sales
* Correlation Heatmap
